---
title: Download
---

You can either download the package using [npm][npm] by running:

```
$ npm install mark.js --save-dev
```

Or using [Bower][bower]:

```
$ bower install mark.js --save-dev
```

Or download the package [manually]({{{defaults.download}}}).

[npm]: http://npmjs.com/
[bower]: http://bower.io/
